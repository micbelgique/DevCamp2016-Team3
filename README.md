## Deployment
Add remote for backend and frontend with:
```
git remote add heroku-backend https://git.heroku.com/cliches-backend.git
git remote add heroku-frontend https://git.heroku.com/cliches-frontend.git
```
And execute batches:
```
heroku-backend.bat
heroku-frontend.bat
```
